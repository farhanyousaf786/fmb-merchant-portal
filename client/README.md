# FMB Merchant Portal

Bare-bones merchant portal for FanMunch delivery system.

## Setup

```bash
npm install
npm start
```

## Features

- ✅ Login page
- ✅ Dashboard with orders view
- ✅ Order management (accept/reject)
- ✅ Responsive design
